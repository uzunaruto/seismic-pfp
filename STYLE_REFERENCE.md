# Seismic PFP — Art Style Reference

## Base Style: Cel-Shaded Anime Illustration

Reference image: `1000413537.jpg` (400x400, analyzed via GPT-4o vision)

### Visual Characteristics

| Aspect | Description |
|--------|-------------|
| **Line art** | Bold, clean outlines (2-3px weight) |
| **Shading** | Flat color zones with 1-2 tone shadows (cel-shading) |
| **Proportions** | Anime-style: larger head (~30% of body), expressive eyes |
| **Hair** | Dynamic, spiky, disheveled — volume and movement |
| **Clothing** | Geometric patterns, fitted, modern streetwear aesthetic |
| **Accessories** | Tech/energy motifs (lightning, glowing elements) |
| **Skin** | Soft neutral tone, minimal texture, clean gradients |

### Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| **Primary dark** | `#1a1a1a` | Hair, clothing base |
| **Skin base** | `#f0d0a0` | Skin tone |
| **Skin shadow** | `#d4a878` | Cel-shade shadow zones |
| **Electric blue** | `#00aaff` | Energy effects, accents |
| **Earth brown** | `#8b6914` | Clothing accents |
| **Highlight** | `#ffffff` | Specular highlights |

### Rendering Rules

1. **NO gradients** on character body — flat cel-shaded zones only
2. **Bold outlines** on all major forms (head, body, limbs)
3. **Single shadow tone** per surface — no smooth blending
4. **Eyes**: Large, expressive, anime-style with catchlight
5. **Hair**: Individual spiky strands, NOT solid mass
6. **Clothing**: Geometric detail, visible folds as angular lines

### Technical Spec

- Canvas: 1000x1000px per layer
- Export: PNG with transparency
- Style: Consistent across ALL trait variations
- Reference: Match cel-shading density and line weight from source

### Anti-Patterns (AVOID)

- ❌ Photorealistic rendering
- ❌ Smooth gradient shading (use flat zones)
- ❌ Thin/sketchy lines (use bold, confident strokes)
- ❌ Western cartoon style (keep anime proportions)
- ❌ Overly cute/chibi (maintain edgy vibe)
